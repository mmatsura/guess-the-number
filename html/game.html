How to Play

Guess the secret number in 5 attempts. After each guess, each digit tile will show a colour hint:

Green — right digit, right position
Orange — right digit, wrong position
Tan — digit not in number

How to Play?

Guess the secret code in a limited number of attempts.

  • After each attempt, you receive hints in the form of colored squares.
  • You have a limited number of attempts and time to guess the code.

Examples:

6421

6 - the number is correct and in the correct position

6421

1 - number is correct but in the wrong position
