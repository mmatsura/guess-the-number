# Guess the Number Game

Логічна гра, де потрібно вгадати закодоване число за обмежену кількість спроб.

# Про гру

Гра "Вгадай число"— це головоломка, де:

- Комп'ютер загадує унікальне число з 4, 5 або 6 цифр
- Ви вводите свій варіант
- Отримуєте підказки кольоровими квадратами:
  - 🟢 **Зелений** — цифра є і на правильному місці
  - 🟠 **Оранжевий** — цифра є, але не на своєму місці
  - ⚫ **Сірий** — цифри немає в загаданому числі

## Технології

- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Frontend**: HTML5, CSS3, JavaScript
- **Authentication**: JWT (JSON Web Tokens)
